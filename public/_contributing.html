Contribute!

When learning to code, you will need to remember lots of new words. So to make the task easier, so here is a collection of some of the key words and their definitions.

We are always looking for people to contribute to this dictionary. If you want to contribute a word and a definition to the dictionary, then please fork this repo and edit the dictionary.yaml file found in data/dictionary.yaml.

For more information, please read the contributing page on Github.
